**🎓 Student Database Management System**

A simple Student Database Management System developed using Python and SQL. This project allows users to manage student records efficiently through database operations like adding, viewing, updating, and deleting student details.

📌 Basic Description

This project is built using Python for application logic and SQL for database management. It performs CRUD (Create, Read, Update, Delete) operations on student records and demonstrates practical implementation of database connectivity.

🚀 Features

Add new student records

View all student details

Search student by ID

Update student information

Delete student records

Database connectivity using SQL

🛠️ Technologies Used

Python

SQL (MySQL / SQLite)

Python Database Connector (mysql-connector / sqlite3)

⚙️ How to Run the Project

Install Python on your system.

Install required database (MySQL or use SQLite).

Create the database using the provided SQL file.

Update database credentials in the Python file (if using MySQL).

Run the program:

python main.py
